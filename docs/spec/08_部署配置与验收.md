# PhotoAI 部署配置与验收

## MVP 服务

```text
postgres
redis
photoai-server
photoai-worker
```

Web 暂不作为 MVP 必需服务。

## 推荐目录

```text
/opt/photoai
├── docker-compose.yml
├── .env
├── data
│   ├── postgres
│   ├── redis
│   └── photoai
│       ├── originals
│       ├── thumbnails
│       ├── previews
│       ├── cache
│       └── uploads
└── logs
```

## Docker Compose 草案

```yaml
services:
  postgres:
    image: pgvector/pgvector:pg16
    environment:
      POSTGRES_DB: photoai
      POSTGRES_USER: photoai
      POSTGRES_PASSWORD: photoai_password
    volumes:
      - ./data/postgres:/var/lib/postgresql/data
    restart: unless-stopped

  redis:
    image: redis:7-alpine
    volumes:
      - ./data/redis:/data
    restart: unless-stopped

  server:
    build:
      context: ../server
    env_file:
      - .env
    volumes:
      - ./data/photoai:/data/photoai
    ports:
      - "8080:8080"
    depends_on:
      - postgres
      - redis
    restart: unless-stopped

  worker:
    build:
      context: ../workers
    env_file:
      - .env
    volumes:
      - ./data/photoai:/data/photoai
    depends_on:
      - server
      - redis
    restart: unless-stopped
```

## 健康检查

接口：

```text
GET /api/health
GET /api/health/db
GET /api/health/redis
GET /api/health/storage
GET /api/health/workers
```

`/api/health` 响应：

```json
{
  "status": "ok",
  "db": "ok",
  "redis": "ok",
  "storage": "ok",
  "version": "0.1.0"
}
```

## 日志

必须记录：

1. 登录失败。
2. Token 刷新失败。
3. 上传失败。
4. 分片缺失。
5. Hash 校验失败。
6. 存储路径不可用。
7. Worker 任务失败。
8. 数据库迁移失败。

日志字段：

```text
timestamp
level
request_id
user_id
module
event
message
extra
```

## 安全要求

1. JWT secret 必须通过环境变量配置。
2. 不允许接口返回服务器绝对路径。
3. 文件下载必须鉴权。
4. 资产查询必须按用户过滤。
5. 上传目录必须防路径穿越。
6. OIDC client secret 不写入日志。
7. Android refresh token 需要可撤销。

## P0 验收清单

### 后端

- [ ] Docker Compose 可启动 PostgreSQL 和 Redis。
- [ ] Server 可连接数据库。
- [ ] Alembic 可执行迁移。
- [ ] 可注册用户。
- [ ] 可登录并获取 Token。
- [ ] 受保护接口无 Token 返回 401。
- [ ] 上传 1 个文件后生成 asset。
- [ ] 同一用户重复上传同 hash 不重复入库。
- [ ] 用户 A 不能访问用户 B 的 asset。

### Android

- [ ] 可连接本地服务端。
- [ ] 可登录。
- [ ] Token 重启后仍可使用。
- [ ] 可申请相册权限。
- [ ] 可扫描照片和视频。
- [ ] 可创建备份任务。
- [ ] 可上传文件。
- [ ] 网络中断后任务保留。
- [ ] 恢复网络后继续上传。

### 媒体处理

- [ ] 上传后创建 thumbnail task。
- [ ] Worker 可领取 task。
- [ ] thumbnail 文件生成。
- [ ] asset_files 写入 thumbnail。
- [ ] Android 可加载缩略图。

## 性能目标

| 场景 | MVP 目标 |
|---|---|
| Android 扫描 | 1000 张照片 30 秒内完成初扫 |
| 上传恢复 | App 重启后 10 秒内恢复队列状态 |
| 后端列表 | 100 张资产分页查询 500ms 内 |
| 缩略图 | CPU 环境稳定生成，不阻塞 API |
| 内存 | 基础服务 16GB 内存长期运行 |
